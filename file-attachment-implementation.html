🎉 KantanPro ファイル添付機能 実装完了

✅ 実装完了! メール送信機能にファイル添付機能が正常に追加されました。

📎 新機能一覧

  • ドラッグ&ドロップ対応 - ファイルをドラッグしてメールに添付
  • ファイル選択UI - クリックによるファイル選択も可能
  • 複数ファイル対応 - 一度に複数のファイルを添付可能
  • ファイル形式制限 - PDF、画像、Office文書、圧縮ファイル等に対応
  • サイズ制限 - 1ファイル10MB、合計50MBまで
  • リアルタイムプレビュー - 選択したファイルの一覧表示
  • 視覚的フィードバック - ドラッグ時のハイライト表示
  • 自動クリーンアップ - 一時ファイルの自動削除

🔧 技術仕様

  • 対応ファイル形式: PDF, JPG, PNG, GIF, DOC, DOCX, XLS, XLSX, ZIP, RAR, 7Z
  • 最大ファイルサイズ: 10MB/ファイル
  • 最大合計サイズ: 50MB
  • セキュリティ: ファイル形式・サイズ検証、一時ファイル自動削除
  • ユーザビリティ: ドラッグ&ドロップ、進捗表示、エラーハンドリング
  • メンテナンス: スケジュール化された一時ファイルクリーンアップ

📝 実装されたファイル

フロントエンド (JavaScript): - js/ktp-email-popup.js (ファイル添付UI、ドラッグ&ドロップ) バックエンド (PHP): - includes/class-ktpwp-ajax.php (ファイルアップロード・添付処理) - ktpwp.php (一時ファイルクリーンアップスケジューラー)

🚀 利用方法

  1. 受注管理画面でメールボタンをクリック
  2. メールポップアップが開く
  3. 「ファイル添付」エリアにファイルをドラッグ&ドロップ
  4. または「クリックして選択」でファイル選択
  5. 添付ファイルを確認してメール送信
